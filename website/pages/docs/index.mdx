# Dashboard Blocks

Dashboard Blocks is a set of components built to create beautiful dashboards

These components are made with composability and ease of use in mind, you should be able to create a dashboard page in less than 10 minutes and with less than 200 lines of code

Do you want to see an example? here is how you build a landing page in 5 minutes
